# bamboofox_website

# How to use
```
git clone https://github.com/bamboofox/bamboofox_website
cd bamboofox_website
npm install
hexo server
```
See it at http://0.0.0.0:4000/

## Hexo-admin

Admin page is at http://0.0.0.0:4000/admin/

# How to deploy
```
hexo generate
hexo deploy
```
See it at http://bamboofox.github.io/
